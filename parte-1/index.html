En nuestro módulo acotaremos la infografía digital a aquellas que se muestran en un navegador web. Para hacer estas infografías es necesario trabajar con HTML, CSS y JavaScript, lenguajes con los que se responde a cada navegador web algunas preguntas.

↑ ↓

1ro. ¿Qué es lo que contiene esta página web?

Esto se responde con HTML (HyperText Markup Language).

HTML es un lenguaje estándar que describe la estructura de las páginas web. HTML5 es la versión más reciente de este lenguaje.

El bloque constructivo más básico del HTML es el elemento. Cada elemento es, generalmente, <etiqueta>contenido</etiqueta>

Así, por ejemplo, un encabezado (heading) de primer nivel (1) se describe como <h1>el texto del título</h1>

↑ ↓

2do. ¿Cómo muestro lo contenido por esta página?

Esto se responde con CSS (Cascading Style Sheets)

CSS es un lenguaje estándar que describe la presentación de las páginas web. CSS3 es la versión más reciente de este lenguaje.

El bloque constructivo más básico del CSS es la regla. Cada regla es, generalmente: selector{propiedad:valor;}

Así, por ejemplo, puedo definir que el encabezado de primer nivel sea rojo h1{color:red;}

↑ ↓

3ro. ¿Qué puede hacer esta página web con su contenido?

Esto se responde con JS (JavaScript).

JavaScript es un lenguaje de programación que controla el comportamiento de las páginas web.

Con JS se pueden escribir secuencias de instrucciones con las que una computadora realizará una tarea determinada. Su estructura puede variar dependiendo de la lógica de cada instrucción, la versión en uso, la library en la que nos apoyemos para resolver algo específico, o el framework de programación con el que se levanta todo el trabajo.

Así, por ejemplo, puedo indicar que al hacer click en este se muestre una alerta.

↑ ↓

CSS y JavaScript pueden incrustarse en un HTML

En un único documento .html se pueden incrustar los lenguajes .css y .js, entre etiquetas de HTML que los anuncien.

↑ ↓

CSS y JavaScript pueden vincularse a un HTML

Uno o varios documentos .html pueden vincularse a un mismo documento .css y también a un mismo documento .js
